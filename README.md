# BitSearch_FrontEnd
BitSearch搜索引擎的前端部分，后端部分见https://github.com/Asiim0v/BitSearch

前端模板参考了项目AnimeHub

前端采用react.js进行开发，运行方式为

```powershell
yarn install
yarn start
```

项目主要包含如下功能

1. 文字搜索网页(采用自行爬取的数据)
2. 文字搜索图片(使用了WUKONG数据集)
3. 搜索提醒
4. 搜索历史
5. 搜索热词
6. 搜索分页
7. 过滤搜索
